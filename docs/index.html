Capabilities

A complete toolkit for computational acoustics -- from simulation to analysis.

3D Visualization

Interactive 3D environment with real-time visualization of acoustic pressure fields. Rotate, zoom, and navigate through the simulation space with intuitive controls optimized for both desktop and mobile devices.

Multiple Microphones

Add and configure multiple microphones with realistic frequency response patterns including omnidirectional, cardioid, and figure-8 polar patterns. Control position and orientation for accurate directional analysis.

Advanced Algorithms

Five propagation algorithms: Standard FDTD, Compact (KW), Interpolated Wideband, Digital Waveguide Mesh, and High-Order O(4,2). Compare them in the benchmark suite.

Multiple Media

Simulate acoustic propagation in air, water, saltwater, or custom media. Support for underwater acoustics with specialized algorithms for aquatic environments.

Geometric Import

Import complex geometries from Blender for realistic architectural and environmental simulations. Define walls, obstacles, and reflective surfaces within the simulation environment.

Comprehensive Analysis

Extensive post-processing tools including source pressure analysis, raw and processed microphone signals, FFT spectra, SPL measurements, and acoustic impedance calculations.

Export Capabilities

Export simulation results in multiple formats including WAV audio files, CSV data, and visualization snapshots. Background processing with downloadable results upon completion.

Preset Scenarios

Pre-configured scenarios for bioacoustics research including cetacean phonation organs and avian syrinx models based on published research for educational and scientific applications.

High Performance

Optimized computation using Web Workers and parallelization strategies for efficient simulation of large domains and extended time periods entirely in your browser.

About the Method

The Finite-Difference Time-Domain (FDTD) method is a well-established numerical technique for solving partial differential equations describing wave propagation. This implementation solves the linearized acoustic wave equations in three dimensions using a staggered-grid leapfrog scheme.

Mathematical Foundation

The simulator solves the coupled first-order acoustic equations relating particle velocity and pressure through spatial and temporal derivatives. The staggered grid approach ensures numerical stability while maintaining second-order accuracy in both space and time.

Stability is guaranteed through the Courant-Friedrichs-Lewy (CFL) condition, with automatic validation and warnings for anti-aliasing to prevent numerical artifacts.

For detailed mathematical derivations, implementation details, and validation studies, please refer to the full documentation, which also includes a Python API reference for running simulations locally.
